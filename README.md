# HexnicAI

**AI-powered tools for smarter learning.**

HexnicAI is a web-based platform that provides intelligent tools designed to help students improve productivity, accessibility, and focus while studying.

---

## Platform Tools

### StudyAI

Smart timetable generator that helps students create balanced study schedules based on subjects and available time.

**Features:**
- AI-based study schedule generation
- Automatic subject distribution
- Time management optimization

### SpeakEasy Tools

Accessibility-focused learning tools designed to assist students with reading and comprehension.

**Features:**
- Text-to-speech support
- Dyslexia-friendly reading interface
- Voice interaction support

### FocusAI (Coming Soon)

A GPT-style AI assistant designed to help students concentrate, research, and learn more effectively.

**Planned Features:**
- AI study assistant
- Question answering
- Learning support tools

## Project Purpose

The goal of HexnicAI is to create a unified learning platform that combines:

- AI-powered productivity tools
- Accessibility support
- Intelligent learning assistance

This project explores practical applications of AI in education and student productivity.
